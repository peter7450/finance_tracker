# Finance Tracker

A full-stack **Expense and Finance Tracking Web App** built with **Django REST Framework (DRF)** for the backend, **React.js** for the frontend, and **PostgreSQL** as the database.  
The app helps users track income, expenses, transactions, and visualize spending habits in real time.
